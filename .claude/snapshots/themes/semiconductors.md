# semiconductors — Theme Snapshot

> Cross-ticker view on `semiconductors`. Journals: `positions/*.md`, `macro.md`
> Last distilled: 2026-06-05

## Theme stance

长期看多定制硅/ARM 抽税链结构性受益;**AVGO 是唯一已持仓的高信念多头**,ARM、MRVL 在 blow-off 估值仅 watch,等回调。

## Drivers

- 定制硅出货增速约通用 GPU 的 3×,结构性吃 NVDA 份额(AVGO $73B AI 订单积压、2027 目标 $100B AI 营收;MRVL XPU/ASIC);深度技术调查证实 AVGO 在旗舰客户、交换(TH6 102.4T)、量产 CPO、3.5D 封装上限领先 ~1 代,但 SerDes(224G)与制程(N2)两家已收窄到平手
- ARMv9 静默提价(每颗版税 ~2× v8)+ AGI CPU(商业模式从 IP 转整芯片,营收 ~10×)
- TSMC N3/N2 节点产能锁死(N3 100% 利用率、demand 3× supply)= 定价权;两家定制硅厂 N2 获取均在第一梯队
- 黄仁勋"光互联(optics where you must)"带动 AI 光链(COHR/LITE/GLW);CPO vs 可插拔光 DSP 路线之争(AVGO CPO 唯一量产、瞄准 MRVL DSP 腿,但 2028–2030 才规模化)

## Risks / invalidation

- ARM、MRVL 估值/微观结构过热,抛物线 + CEO 喊单驱动而非利润驱动
- 定制硅毛利重定价:AVGO 财报后盘后 -14%,"ASIC 放量稀释毛利"(代设计、无 GPU 式定价垄断)从个股升级为子板块问题
- 客户洗牌:微软 Maia 定制芯片设计据报正从 MRVL 谈判转向 AVGO
- 数据中心 royalty YoY 增速跌破 +50% / AGI CPU 出货滑期 / SoftBank 系统性减持(ARM invalidation)
- TSMC N3/N2 allocation 被削减或新厂延期;AI ROI 退潮、定制硅订单不兑现

## Names exposed

### Long
- **AVGO** (years, high) — 定制硅旗舰;技术面证实最赚钱环节(旗舰客户 ~70% 份额、交换 ~1 代、唯一量产 CPO、封装上限)真领先;已持仓 ~$410

### Watching
- **ARM** (months, med) — AI 算力抽税 + AGI CPU(IP→整芯片,营收 ~10×);ARMv9 静默提价 ~2×;blow-off 估值,等 $260–290(MA50)回调
- **MRVL** (obs only) — 定制硅 XPU/ASIC 老二;SerDes/制程已与 AVGO 平手、光 DSP(Inphi)当下反领先,但旗舰客户被挖、量产 CPO/交换落后 ~1 代;入场抑制器

## Macro context

2026-05-30 / 06-04 Macro —— 定制硅(ASIC)结构性侵蚀 NVDA GPU 份额是核心轮动主线;领导权从在位者(NVDA)轮向第二层定制硅架构商(AVGO/MRVL)。但 AVGO 财报后子板块杀跌(MRVL -7.7% vs NVDA -1.1%)提示:定制硅天生毛利低于商用芯片,市场已开始在 AI 内部按毛利/估值挑肥拣瘦。
