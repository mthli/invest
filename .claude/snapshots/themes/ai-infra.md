# ai-infra — Theme Snapshot

> Cross-ticker view on `ai-infra`. Journals: `positions/*.md`, `macro.md`
> Last distilled: 2026-06-04

## Theme stance

Watching — 长期看多 AI 算力/定制硅建设,但每个名字都因估值/微观结构过热而仅 watch,等回调入场。

## Drivers

- 超大盘 2026 AI capex 合计逼近 $700B(AMZN ~$200B、GOOGL $180–190B),需求端已锁 OpenAI ~2GW + Anthropic 最高 5GW(2027 起)
- 定制硅(ASIC/XPU)出货增速约通用 GPU 的 3×,结构性侵蚀 NVDA 份额(AVGO、MRVL、GOOGL TPU、AMZN Trainium)
- ARM 化不可逆:Graviton/Axion/Cobalt/Grace + Nvidia RTX Spark/Vera 每颗付 ARM 版税,新增 AI PC 版税腿
- 领导权从在位者(NVDA)轮向第二层受益者(定制硅)与已证明变现者(MSFT)

## Risks / invalidation

- 估值/微观结构过热:ARM 站上均值目标 ~70%、forward ~135–160x;MRVL forward 47.6x、站上目标 31%;薄流通盘 +"小消息→大波动"= blow-off top 风险
- capex ≥ 经营现金流:FCF 顺风被吞(AMZN FCF 转负、GOOGL 净发行稀释),未来巨额 D&A 压缩报表 EPS
- 市场对 AI ROI 失去耐心,按跑率收入而非"建多少"打分 → 纯 capex 受益者重定价风险
- 需求兑现转弱:数据中心 royalty 增速放缓 / AGI CPU 出货滑期 / hyperscaler 回退 x86(ARM 专属)

## Names exposed

### Watching
- **ARM** (months, med) — 从 IP 公司重分类为"AI 算力抽税器 + 数据中心整芯片";现价比均值目标高 ~70%、forward ~135–160x,等 $260–290(MA50)回调
- **DELL** (months, med) — ai-infra 里估值最低但盈利质量最低的"算力组装与交付商";Q1 FY27 blowout 跳空后重估到 ~25x,便宜逻辑用尽,等跳空消化
- **MRVL** (obs only) — 定制硅(XPU/ASIC)真实,但黄仁勋"万亿"喊单后 +275%/3mo、站上目标价 31%,入场抑制器
- **AMZN** (obs only) — capex ~$200B、FCF 转负;Trainium 定制硅 + 需求已锁(OpenAI/Anthropic);痛点财务型(随 capex 见顶可自愈)
- **GOOGL** (obs only) — $80B 史上最大股权融资投 TPU;痛点商业模式型(AI 搜索侵蚀广告核心,不可自愈)

## Macro context

两条 Macro(2026-05-30 / 2026-06-03)框定:AI 交易进入中段,从"capex beta"转向"变现证明 + 结构性份额转移"的分化选股;且 capex 已系统性超过超大盘经营现金流(撞墙——GOOGL 对外稀释、AMZN 烧 FCF)。看多算力/定制硅链条不变,但超大盘本体打分轴转向变现转化率 + 稀释 + 折旧;囚徒困境下有过度建设风险。
