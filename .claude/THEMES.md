# Themes Registry

This file lists the canonical theme IDs used across all Thesis and Observation blocks.
Themes are the lateral axis that lets `/distill-theme` aggregate views across tickers.

Naming constraints: ASCII only, lowercase, hyphens for word separation. Keep the registry
to ~15-25 entries — a bloated registry erodes the value of `/distill-theme`.

## Registered themes

- `ai-infra` — AI 基础设施层:数据中心 host CPU、AI 加速器(GPU/TPU/ASIC)、网络互联、hyperscaler 资本开支周期
- `semiconductors` — 半导体行业:代工产能、芯片设计 IP、节点演进(N3/N2)、CPU/GPU 架构竞争
